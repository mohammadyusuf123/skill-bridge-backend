                        INTEGRATION WITH BACKEND
================================================================================

Frontend designed for Skill Bridge Backend API.

Backend expected on: http://localhost:3001 (development)

Frontend uses these API endpoints:
  Authentication: /auth/sign-up, /auth/sign-in, /auth/sign-out
  Users: GET /api/users/profile, PUT /api/users/profile
  Tutors: GET/POST /api/tutors, GET /api/tutors/search
  Bookings: POST/GET /api/bookings, PUT /api/bookings/:id
  Categories: GET /api/categories
  Availability: GET/POST /api/availability
  Reviews: GET/POST /api/reviews
